# Planning Framework (4-Hour MVP)

## In Scope
- Employee clock-in
- Employee clock-out
- Punch validation
- Duplicate punch prevention
- Shift history or today summary
- Audit log entries
- Payroll-ready summary endpoint

## Out Of Scope (Deferred)
- Leave workflows
- Scheduling and break enforcement
- Full payroll rules (overtime/holiday/night differential/PTO/comp time)
- Full compliance and enterprise RBAC depth

## Delivery Order
1. Backend entities and API contracts
2. Backend validation and audit logging
3. Thin frontend flow
4. API and UI tests
5. Security and demo checklist
