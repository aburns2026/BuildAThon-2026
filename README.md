# BuildAThon-2026

BuildAThon MVP workspace organized for delivery artifacts and implementation code.

## Structure

- docs/artifacts/
  - planning and execution documents for MVP scope control
- artifacts/playwright-runs/
  - generated Playwright screenshot evidence by run id
- code/
  - backend/
  - frontend/
  - tests/api/
  - tests/e2e/

## Quick Start

1. Lock MVP scope in `docs/artifacts/planning-framework.md`.
2. Define API contract in `docs/artifacts/requirements-traceability-matrix.md` and backend notes.
3. Implement backend first, then frontend, then tests.
